JavaScript Grammar

Statements:

const message = 'Hello World!';

alert(message);

Block:

A block is a series of statements that are collected together inside curly braces

Blocks do not need to be terminated by a semicolon.

Data Types

  1. String
  2. Symbol
  3. Number
  4. Boolean
  5. Undefined
  6. Null
  7. Object

typeof operator

A operator for finding out the type of a value

Variables

From ES6 onwards, JavaScript uses the keywords const and let to declare variables.

They are block scoped, so their value only exists inside the block they are declared in.

Global Scope

Any variable declared outside of a block is said to have global scope.
